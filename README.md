# 💫 About Me:

I'm a passionate young developer with a strong interest in **web development, automation, and server management**.  
I enjoy exploring how technology works at a deeper level — from setting up containers to optimizing backend systems — and I'm constantly learning to improve both my technical and problem-solving skills.

- Developing and deploying **self-hosted solutions** using Docker and Linux  
- Building efficient **web applications and backend systems**  
- Exploring **networking, DNS management, and system automation**  
- Learning about **modern authentication systems** like JWT and Auth0

### 🚀 Featured Projects

- **Automation Scripts** - Created Python and Bash scripts for system optimization and task automation
